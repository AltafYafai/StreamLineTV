## ✨ Features

- **Beautiful Material You Design**: Soft shapes, organic streams, and beautiful pastel palettes designed for modern displays.
- **TV-First Experience**: Fully optimized for D-Pad navigation and living room viewing distances.
- **Stremio Addon Ecosystem**: Bring your own content. StreamLineTV acts as a clean interface to discover and watch your media using user-installed extensions.
- **Advanced Playback**: Integrated ExoPlayer/Media3 for robust codec support, AFR (Auto Frame Rate), and PiP (Picture-in-Picture).

## 🚀 Getting Started

Currently in active development. Releases will be available in the **[Releases](#)** tab once stable.

### Prerequisites to Build
- Android Studio (latest)
- JDK 17+
- Android SDK (API 33+)

```bash
git clone https://github.com/AltafYafai/StreamLineTV.git
cd StreamLineTV
./gradlew assembleDebug
```

## ⚖️ Legal & Disclaimer

StreamLineTV functions strictly as a client-side interface. It does **not** host, store, scrape, or distribute any media content. It merely provides a beautiful layout for content you own or access legally through third-party user-installed extensions. We are not affiliated with any third-party content providers.
